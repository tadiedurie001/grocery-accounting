# Grocery Shop Accounting System

A Flask-based web application for managing grocery shop accounting operations.

## Features

- Product management
- Transaction tracking
- Financial reporting
- RESTful API
- Database integration

## Technology Stack

- Python 3.8+
- Flask
- SQLAlchemy
- PostgreSQL
- Bootstrap
- jQuery
